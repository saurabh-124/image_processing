# image_processing

This repo is for image processing, 6th sem, Btech.

## How to run

1. install python venv

```bash
python -m venv myenv
```

2. activate myenv

- windows
```bash
myenv\Scripts\activate
```

- linux
```bash
source myenv/bin/activate
```

3. install the requirements

```bash
pip install -r requirements
```

4. run the code

```bash
python <filename>
```
